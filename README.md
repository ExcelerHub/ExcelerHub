# 🚀 ExcelerHub

### Learn • Grow • Connect

ExcelerHub is a modern **Learning & Internship Management Platform** built with **Flutter**. It provides a seamless experience for learners to discover programs, enroll in opportunities, track progress, manage tasks, stay updated with announcements, submit feedback, and maintain their learning profile through a clean and user-friendly interface.

## 🔄 Enrollment-Based Workflow

### Before Joining a Program
Users can:
- Browse Programs
- View Program Details
- Manage Profile

Users cannot:
- Access Tasks
- Track Progress
- Submit Feedback

---

### After Joining a Program
Users can:
- Access Learning Tasks
- Track Progress
- Submit Feedback
- Receive Program Updates

---

### After Leaving a Program
The system automatically:
- Removes Associated Tasks
- Resets Program Progress
- Revokes Feedback Access
- Returns User to Non-Enrolled State
