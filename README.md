# 🚀 ExcelerHub

### Learn • Grow • Connect

ExcelerHub is a modern **Learning & Internship Management Platform** built with **Flutter**. It provides a seamless experience for learners to discover programs, enroll in opportunities, track progress, manage tasks, stay updated with announcements, submit feedback, and maintain their learning profile through a clean and user-friendly interface.

## ✨ Features

- User Registration & Login with session persistence
- Browse & search internship programs
- Enroll and leave programs
- Enrollment-based task tracking
- Announcements & updates
- Feedback form with star ratings
- User profile management

---

## 🛠 Technology Stack

- **Frontend:** Flutter & Dart
- **Data:** JSON file (Mock API via MockApiService)
- **Local Storage:** shared_preferences (login session persistence)
- **Architecture:** Service layer with singleton pattern
- **Tools:** VS Code, Git, GitHub

---

## ▶️ Getting Started

### 1. Clone the repository
git clone https://github.com/ExcelerHub/ExcelerHub.git
cd ExcelerHub

### 2. Install dependencies
flutter pub get

### 3. Run the app
flutter run -d chrome
